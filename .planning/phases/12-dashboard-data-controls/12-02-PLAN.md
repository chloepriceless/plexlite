Create `dvhub/public/components/dashboard/control-compute.js` with pure functions ported from old app.js lines 729-751:

1. **`createPendingState({ currentReadback, submittedValue, submittedAt })`**
   - Returns `{ previousReadback: currentReadback, targetValue: submittedValue, submittedAt: submittedAt || Date.now() }`

2. **`resolvePendingState({ pendingState, readbackValue })`**
   - If `!pendingState` return null
   - If `readbackValue == null` return pendingState (still pending, no readback yet)
   - If `readbackValue === pendingState.targetValue` return null (confirmed)
   - If `readbackValue !== pendingState.previousReadback` return null (changed by something else)
   - Otherwise return pendingState (still waiting)

3. **`computeRenderState({ readbackValue, pendingState })`**
   - Calls resolvePendingState internally
   - Returns `{ pendingState: nextPendingState, shouldBlink: Boolean(nextPendingState) }`

All functions use `export function` (ES module exports).

- Test resolvePendingState returns pendingState when readbackValue is null
