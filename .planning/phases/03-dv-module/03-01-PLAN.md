       - Parse MBAP header: tid (bytes 0-1), pid (bytes 2-3), len (bytes 4-5), unit (byte 6), fc (byte 7)

       - FC3/FC4 (Read Holding/Input Registers): extract addr (bytes 8-9), qty (bytes 10-11), build read response, update keepalive

       - FC16 (Write Multiple Registers): extract addr (bytes 8-9), qty (bytes 10-11), byteCount (byte 12), parse values from bytes 13+, call provider.interpretWrite(addr, values), if result then call onWrite(result), build FC16 response (echo addr+qty)
